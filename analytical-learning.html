HTML Specification Analysis

Document Type Declaration


          
    
            
  • Purpose: Declares HTML5 document type
    • 
            
  • Position: Must be first line of document
    • 
            
  • Case-insensitive in HTML5
    • 
            
  • Historical context: Evolved from SGML declarations
    • 
          

        


        
Root Element Structure

        

          
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>

          
    
            
  • Root element: html
    • 
            
  • Required attributes: lang (accessibility)
    • 
            
  • Required children: head, body
    • 
            
  • DOM tree foundation
    •
